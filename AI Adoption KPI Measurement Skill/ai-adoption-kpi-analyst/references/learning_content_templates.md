## Prompt Templates

### 模板 1: 专业写作助手

```markdown
你是[角色]，负责[任务类型]撰写。

## 任务
[具体写作任务描述]

## 背景信息
[相关背景、目标受众、用途]

## 要求
1. [格式要求]
2. [风格要求]
3. [字数要求]

## 输出格式
[期望的输出结构]

请生成[具体内容]，确保符合以上要求。
```

**适用场景**: 邮件撰写、报告生成、文案创作、文档起草

**使用示例**:
```markdown
你是资深产品经理，负责撰写产品需求文档。

## 任务
为新功能"智能推荐"撰写 PRD

## 背景信息
- 目标用户：企业版客户
- 使用场景：首页内容推荐
- 现有系统：基于规则的推荐引擎 v1.0

## 要求
1. 包含功能描述、用户故事、验收标准
2. 风格专业、结构清晰
3. 字数 800-1200 字

## 输出格式
1. 功能概述
2. 用户故事（3-5条）
3. 验收标准
4. 技术约束

请生成完整的 PRD 文档。
```

## Micro-Tutorials

### 微教程 1: 结构化提示词

```
结构化提示词 = 角色(Role) + 任务(Task) + 上下文(Context) + 输出格式(Output)

核心要点：
1. Role: 明确 AI 应该扮演什么角色
2. Task: 清晰描述要完成的任务
3. Context: 提供必要的背景信息
4. Output: 指定期望的输出格式

示例：
Role: 资深产品经理
Task: 撰写功能需求文档
Context: 针对企业版用户，优化推荐算法
Output: Markdown 格式，包含用户故事和验收标准
```

### 微教程 2: 上下文构建

```
好上下文 = 背景 + 约束 + 示例

构建方法：
1. 背景：谁需要？解决什么问题？
2. 约束：有哪些限制条件？
3. 示例：类似的好例子是什么样的？

避坑：
- 不要假设 AI 知道你的工作
- 不要省略关键背景信息
- 不要只说"帮我写"就结束
```

### 微教程 3: 迭代优化工作流

```
迭代工作流：
1. 初稿请求：给出基本要求和格式
2. 反馈：明确指出需要调整的部分
3. 优化：基于反馈再次生成
4. 验证：检查是否符合要求

关键技巧：
- 每次反馈要具体（指出具体问题）
- 不要一次性要求太多修改
- 记录有效提示词模式
```

### 微教程 4: 复杂任务拆解

```
拆解原则：
1. 识别主要目标
2. 分解为子任务
3. 确定执行顺序
4. 分别处理每个子任务
5. 整合最终结果

示例：撰写市场分析报告
├─ 子任务 1: 收集竞品信息
├─ 子任务 2: 分析市场份额
├─ 子任务 3: 用户调研总结
├─ 子任务 4: SWOT 分析
└─ 子任务 5: 整合成报告
```

### 微教程 5: 效果验证

```
验证清单：
1. 准确性：信息是否正确？
2. 完整性：是否覆盖所有要求？
3. 可用性：能否直接使用或需修改？
4. 格式：是否符合要求的格式？

提升方法：
- 建立个人验收标准
- 记录常见问题
- 迭代优化提示词
- 积累成功案例
```

## Success Metrics

### 初级用户成功指标

| 周次 | 指标 | 目标值 | 验证方式 |
|------|------|--------|----------|
| Week 1 | 任务完成率 | >= 70% | 任务提交记录 |
| Week 2 | 结构化提示词使用率 | >= 50% | 提示词分析 |
| Week 3 | 效率提升 | >= 15% | 时间对比 |
| Week 4 | AI 使用频率 | 每天 >= 2 次 | 使用日志 |

### 中级用户成功指标

| 周次 | 指标 | 目标值 | 验证方式 |
|------|------|--------|----------|
| Week 1 | 输出修改率 | <= 30% | 修改记录 |
| Week 2 | 复杂任务完成数 | >= 3 个 | 任务记录 |
| Week 3 | 个人模板库数量 | >= 5 个 | 模板收集 |

### 高级用户成功指标

| 周次 | 指标 | 目标值 | 验证方式 |
|------|------|--------|----------|
| Week 1 | 高级技巧应用率 | >= 40% | 技巧使用统计 |
| Week 2 | 团队辅导次数 | >= 3 次 | 辅导记录 |

### KPI 达标标准

| KPI | 初级目标 | 中级目标 | 高级目标 |
|-----|----------|----------|----------|
| Usage | >= 60% | >= 75% | >= 85% |
| Productivity | >= 15% | >= 25% | >= 35% |
| Confidence | >= 50% | >= 70% | >= 85% |
| Proficiency | >= 60% | >= 75% | >= 85% |
